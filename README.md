Customer Segmentation & Churn Prediction
End-to-end customer analytics pipeline built using the
Kaggle Bank Customer Churn dataset (10,000 records).
What This Project Does

Performs full EDA on 10,000 bank customer records
Applies K-Means clustering to identify 4 customer segments
Builds a churn prediction model (Random Forest, 86% accuracy)
Generates a 6-panel executive dashboard
Delivers a written insights report with 4 recommendations
